npx ironoutline COMMAND

COMMAND:

  json2csv  Generates a CSV to import into a spreadsheet to manipulate the course outline
  csv2json  Generates JSON outline for md2oedx from a CSV.
  
json2csv
===

Generates a CSV to import into a spreadsheet to manipulate the course outline

Usage:

  npx ironoutline json2csv JSON_PATH

Example:

  npx ironoutline json2csv ft.json

Arguments:

  JSON_PATH:  json file path

csv2json
===

Generates JSON outline for md2oedx from a CSV.

Usage:

  npx ironoutline csv2json PTFT "CSV_URL"|CSV_PATH [ --tzid=TZID --start=START_DATE --hollidays=HOLLIDAYS ]

Example:

  npx ironoutline csv2json pt "${csvUrlOrPath}" --start=2020-06-02 --hollidays=2020-06-20,2020-07-04,2020-07-14,2020-08-11,2020-08-13,2020-08-15,2020-08-18,2020-08-20,2020-08-22,2020-09-19,2020-10-17,2020-11-10,2020-11-21

Arguments:

  PTFT:       ft or pt (Default: ft)
  
  CSV_URL:    A CSV URL where the outline is configured (Default: https://docs.google.com/spreadsheets/d/e/2PACX-1vSPb9g-3UgLBIrjBekCEppZ7k733mCQehR9S3OZBxafwQEuXsxkAzC4VkSzOStT6b0Dc851CyLUOc2i/pub?gid=0&single=true&output=csv)
  CSV_PATH:   A CSV file path

Options:

  TZID: The timezone name where bootcamp is located (see: https://en.wikipedia.org/wiki/List_of_tz_database_time_zones)

  START_DATE: The day in the format YYYY-MM-DD when the course begins

  HOLLIDAYS:  A comma-separated list of days in the format YYYY-MM-DD for days-off
